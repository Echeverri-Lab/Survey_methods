# Quality Control

This folder contains checklists and logs for multi-stage survey QC.

## Files

- `qc_checklist.md` — Five-stage checklist covering:
  - **Stage 1**: Internal pre-pilot review (matrix completeness, item wording, response options, order, platform)
  - **Stage 2**: Cognitive interview protocol and issue log
  - **Stage 3**: Pilot study protocol (timing, dropout, attention checks, psychometrics, open-ended content)
  - **Stage 4**: During data collection (ongoing spot-checks)
  - **Stage 5**: Post-collection data cleaning

## How to use
Complete the checklist in three dated passes and save each as a dated copy (e.g., `qc_checklist_v1_2025-09-01.md`). Retain all three passes in this folder for audit purposes.
